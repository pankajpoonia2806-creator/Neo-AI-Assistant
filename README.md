# 🤖 Neo AI Assistant

Neo AI is a personal AI assistant built in Python.

## Features

- 🧠 Smart Memory
- 💬 Chat History
- 🌐 Internet Search
- 🌦 Weather
- 📚 Wikipedia
- 🖥 PC Control
- 🔊 Voice Reply
- 💾 SQLite Database

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
